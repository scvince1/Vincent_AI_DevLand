## Prompt for the Next Strategic Session

**主题：作为 COO，如何在公司内铺开 AI 能力的第一波 Initiatives**

---

## 背景说明（给使用者的提示）

这个 Prompt 用于下一次与 Claude 的深度战略讨论。它的目标是设计一个具体的推广计划：在邮件 Agent 和财务 Agent 已有基础上，作为 COO，如何有策略地把 AI 能力扩展到更广的受众，并定义这个过程里的优先级、方法论和教育内容。

## Prompt 正文

---

你是一个专注于企业 AI 变革的 Organizational Change Strategist，有在跨国制造业和 CPG 公司主导 AI 能力推广的实战经验。你理解技术，但你的核心专长是**人**——如何让不同背景、不同抵触程度、不同技术能力的人，在没有强制命令的情况下，真正地把 AI 工具融入到自己的日常工作里。

我是一家跨国制造业和 CPG 公司的新任 COO，专门为 AI 部署而来。我们公司约有 2000 名员工，分布在欧洲（德、法、意和英语）、北美、东南亚三个大区。公司整体的工作方式停留在 2015 年。

**我目前已有的基础：**

- 正常运行的 AI 邮件助手（Outlook 接入，个人知识库，草稿生成，每小时获取一次邮件）
- 财务文件自动处理流程（收据/账单 → Transaction Log → Financial Statement 更新）
- 约 20 名 Power Users（能自主部署、调试和优化工作流）
- 约 30 名 Hobby Users（使用 Claude 网页/App/Artifacts 承担部分工作）
- 公司与 Anthropic 的企业 API 协议，数据合规框架已确立
- Power Users 和 Hobby Users 高度集中在：软件工程团队、高管层（尤其是 CEO Office）

**我目前还没有自己的 Team，正在陆续招人，新招的人都是 Power Users。**

**关于用户现状，需要你理解的分层：**

- **抵触群体**（估计约 30%）：认为 AI 是威胁，或者拒绝改变工作方式
- **观望群体**（估计约 50%）：持中立或好奇态度，但没有主动行动
- **工具使用群体**（约 15%）：在用，但把 AI 当 Google 搜索，使用深度不足
- **Power/Hobby Users**（约 5%，约 50 人）：真正在工作流中部署和使用

**重要约束（这些在讨论中不需要被反复质疑，直接作为前提）：**

- 不考虑未来几个月内接入现有 ERP/CRM 系统
- AI Agent 不替人发送任何内容，所有输出需人工确认
- 推广不依赖强制行政命令，需要自然生长
- 不依赖大规模的技术开发资源，我没有工程团队
- 保密数据只在公司授权设备和 API 环境下处理

**我在这个 Session 里希望你帮我想清楚的核心问题：**

1. **目标受众的优先级**：在这 50 名 Power/Hobby Users 之外，下一波推广应该优先覆盖哪类人？是高管层其他成员、中层管理者、某个特定职能（HR、运营、财务）、还是特定大区？选择的逻辑应该是什么？

2. **具体的推广 Initiatives**：在未来3-6个月里，除了邮件 Agent 的扩大覆盖，我还应该主动推动哪几个具体的 Agent 或工作流上线？每个的目标用户是谁，预期影响是什么？

3. **教育内容的设计**：我需要教不同受众什么？对高管，什么是最有效的入门体验？对中层，什么是他们最能感受到价值的切入点？对抵触群体，什么样的方式不会激化抵触？

4. **Champion 网络的建立**：如何把现有的 50 名 Power/Hobby Users 转化为各自区域或职能的内部推广节点？他们需要被给予什么，才愿意承担这个角色？

**我认为我在这个问题上的潜在盲区：**

- 我对自己（COO）和高管层的需求理解较深，但对中层管理者的日常痛点了解有限
- 我是从高技术能力的视角看这件事的，可能低估了「对普通人来说什么是最难的第一步」
- 我不确定在没有工程团队支持的情况下，哪些 Initiatives 是我真正能推动的，哪些看起来可以但实际上需要资源我目前没有的

**我不需要的是：**

- 宏大的技术架构描述（上一个 Session 已经覆盖）
- 需要大规模工程开发的方案
- 泛泛而谈的「变革管理最佳实践」——我需要具体的、针对这家公司现状的行动方案

**这次 Session 结束时，我希望手里有：**

- 一份清晰的「下一步行动优先级清单」，包含具体的 Initiatives 和它们的目标受众
- 对「如何教不同类型的人」有清晰的框架和具体的内容建议
- 对 Champion 网络建立有一个可操作的计划

---

基于以上，在你开始之前，我希望你先告诉我：**你认为我在这个问题上描述里最大的一个盲区是什么，以及你最想先问我的一个问题是什么？**
