## I. 九阳集团全景

### I.1 九阳是什么 (零基础速读段)

九阳股份有限公司 (深交所 002242), 1994 年王旭宁在山东济南创立, 发明全球首台家用全自动豆浆机 (30 万元起步, 4 人团队)。2008 年深交所上市 (国泰君安承销)。核心品类为营养煲 (豆浆机 / 破壁机 / 电饭煲)、食品加工机、西式电器 (空气炸锅等) 三大系列, 2024 年占营收 93%。中国厨房小家电龙头之一, 尤以豆浆机 + 破壁机见长。

当前状态: 2020 营收峰值 112.24 亿、净利 9.40 亿; 2025 营收降至 82.10 亿、净利 1.18 亿, 连续五年双降。境外收入 2025 为 8.25 亿 (-48.83%), 占比从 18% 降至 10%, 主因是美国关税 + SharkNinja 代工腰斩。

控股结构: 王旭宁通过上海力鸿 (50.13%) + JS Global Capital (16.94%) 合计实控 67%+。2022-12 王旭宁本人退出 A 股董事会, 转任副董事长 (非执行) + JS Global 主席兼 CEO。

## II. SharkNinja × 九阳 历史渊源

### II.1 核心关系速读段

2017 年 7 月, 王旭宁通过个人载体联合 CDH 投资 (鼎晖) 收购 SharkNinja (原美国 / 加拿大公司, 1998 Shark 品牌加拿大 Euro-Pro 创立, 2009 推出 Ninja), 对价约 15 亿美元 / 100 亿人民币 (具体金额未在 SEC 或港交所文件直接披露)。同年 10 月 JS Global Lifestyle 控股公司成立以统一管理九阳 + SharkNinja。2019 年 JS Global 港股 1691.HK 上市。**2023-07-31 SharkNinja 从 JS Global 分拆, 直接挂牌 NYSE 独立上市**, 估值约 42 亿美元, 同日签三协议 (品牌许可 20 年 / 采购 / 联合开发)。

**当前两家关系的本质**: 法律框架存续 (品牌许可至 2043), 董事会层面家族交叉 (王旭宁任 SN Chairman + JS Global CEO + 九阳副董事长; Barney Wang 2024-07 入 SN 董事会), 但业务层面实质关联快速归零: SharkNinja 2025 年把美国销量的 90%+ 产能迁出中国至越南/印尼/马来/柬埔寨, 对九阳采购季度从 1769 万美元降至 240 万美元 (-87%), 2026 年品牌许可交易上限仅 2800 万美元, Sourcing Service Fee 合同 2025-07-31 终止。

## VII. 跨事实连接性分析 (6 段)

### VII.1 九阳技术叙事 vs 执行层的断层

**事实**: R&D 连降 (3.90 → 3.89 → 3.61 → 3.03 亿) + AI 团队 <5 人 + 数据中台 2022 后无公开更新 + 香港创新中心 2023-12 宣布但 2026 进展不明 + 深思考 7.92% 持股无产品整合进展。

**连接**: 五个事实指向同一结构: 技术叙事层保持抱负密度, 但执行资源层持续收缩, 方向背离已持续三年以上, 非短期调整。R&D 绝对值连降说明不是营收被动稀释, AI 团队 <5 人说明九阳没有能主导 AI 系统工程的内部能力单元。数据中台 / 深思考 / 香港中心都是"宣布后静止"状态。这一落差在五年业绩压力期内持续扩大, 而非收窄。

### VII.2 两家公司实质业务关系的归零进程

**事实**: SN 2025 迁产能出中国 + 九阳代工 2024 Q1 1769 万美元 → 2025 Q1 240 万美元 (-87%) + 2025-07-31 Sourcing Service Fee 合同终止 + 2026 品牌许可上限仅 2800 万美元 + 2025-10 九阳境外改授权经销模式 (JS Global 成主要经销商)。

**连接**: 采购腰斩 + 合同终止 + 上限低 + 九阳境外模式转为依附 JS Global 渠道, 五个信号叠加说明实质经济关联已系统性收缩, 速度快于 2023 分拆预设。剩余联系主要是法律框架 (品牌许可至 2043) + 董事会家族交叉 (王旭宁 + Barney)。法律框架存续和业务实质关联是两个不同层面, 当前业务层面正在快速归零。

### VII.3 九阳战略模式的一致性观察

**事实**: 2022-12 换帅 + 2023-12 宣布香港 2 亿美元 + 2024 甲壳虫清退 + 2025-05 骨折价持股疑云 + 2025-05 郭浪离任 + 2025 五连降 + 2025-10 境外改授权经销。

**连接**: 三年内密集的人事调整 + 战略宣布 + 清退失败 + 权代过渡, 呈现的模式不是系统性转型, 而是每次应对下行压力时, 典型动作是调整人事或宣布更大格局战略投入, 而不是在现有边界内做持续性执行迭代。总经理从外部高薪空降 (郭浪 492 万, 品牌导向) 切回财务内部过渡 (阚建刚, JS Global 系), 是资源收紧 + 风险偏好下降信号。战略方向调整频率高于一个战略得以充分执行所需时间。

### VII.4 九阳出海的结构性矛盾

**事实**: 境外毛利率 8.60% vs 境内 29.77% (差 21pp) + 代工通道腰斩 + "出海"战略方向 + 历史出海依赖 SharkNinja 代工无独立品牌出海体系 + 2025-10 改授权经销 (依附 JS Global)。

**连接**: 21pp 毛利差说明出海每元营收贡献不到境内三分之一, 结合历史通道依赖 SN, 最合理解释是"定价权和渠道控制不在九阳手里, 毛利压缩是路径依赖的结构性结果"。代工通道萎缩直接带来境外收入 -48.83%, 两者量级吻合。2025-10 改授权经销更进一步将境外终端销售控制权让给 JS Global。"独立品牌出海"所需渠道基础设施在公开信息中没有建设记录, 是需要数年和大量资源的重建工程。

### VII.5 家族控制重心的外移进程

**事实**: 王旭宁 2022-12 退九阳 A 股董事会 + 现任 JS Global 主席 CEO + SN Chairman (~38.7%) + Barney Wang 2024-07 入 SN 董事会 + 2024-12 JS HK 增资九阳香港 10% + 王旭宁 2024 薪酬 5.21 亿港元 (JS Global / SN 层面, 非九阳 A 股) + 2025-06 王旭宁将 700 万 SN 股 (4.97%) 投票权处置权转给美国居民管理的 Delaware LLC, 保留经济权益。

**连接**: 家族对九阳 A 股的控制通过实控人结构 (67%+) 完整, 但家族成员直接执行精力 / 薪酬流向 / 第二代接班布局 (Barney 入 SN 董事会非九阳), 都集中于美股 + 港股层面。2024-12 JS HK 直接介入九阳香港技术载体, 形成与 A 股平行的参与路径。2025-06 部分 SN 投票权转 Delaware LLC 进一步优化美股层面的治理。对九阳 A 股独立股东意味着: 控制权完整但最优先的管理注意力分配在集团上层, 不是 A 股子公司。

### VII.6 九阳技术负责人所处组织环境

**事实**: 前五段综合。

**连接**: 技术负责人当前所处环境可从三维度描述:
- **资源供给**: R&D 占营收 3.69% vs 销售费用率 17% (四倍差), AI 团队 <5 人; 在压力下研发是被压缩方
- **战略地位**: 最大公开技术承诺香港中心 10% 股权在 JS Global HK 手里, 定位中长期研究而非当期能力建设; 技术负责人与香港中心的架构关系在公开信息中无清晰描述
- **话语权**: 阚建刚代行总经理 (财务背景, JS Global 系), 决策层倾向财务管控风险收缩, 技术路线推进需要在 ROI 明确前说服财务导向决策层

结构性背景: 九阳 A 股在集团 (A 股 + 港股 + 美股) 内的资本吸引力明显低于 SharkNinja, 集团最活跃的技术叙事与资本注意力在 SN 层面。技术负责人所在的是资源收缩期 + 决策层财务谨慎 + 集团资本中心在上方另一主体的组织位置。
