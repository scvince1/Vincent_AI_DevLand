## Q1：如何关掉或删掉这个 Hook

**完全删除：** 用任意文本编辑器打开 `%USERPROFILE%\.claude\settings.json`，找到并删除 `SessionStart` 和 `SubagentStart` 下对应的 hook 条目，保存即可。改完重启 Claude Code 生效。

**临时禁用（不删文件）：** 把 `settings.json` 里 hook 命令改成无害命令，或者在条目里加一个 `"disabled": true`（Claude Code 不一定支持，但直接注释掉 JSON 会导致解析失败——所以最安全的临时方案是把 hook 条目整行删掉，需要时再粘贴回去）。

实际上最快的操作：直接删掉 `inject_datetime.js`（或 `.py`）文件本身——hook 命令找不到文件会静默失败，不影响 Claude Code 正常使用，相当于软禁用。
