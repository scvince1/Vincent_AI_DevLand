# Concurrent Training for Multi-Sport Recreational Athletes

## 1. Interference Effect: Less Scary Than Previously Thought

- 2024 meta-analysis (Medicine, Dec 2024): concurrent training does NOT significantly reduce maximal strength or hypertrophy vs. resistance training alone
- Interference primarily affects explosive strength/power, not hypertrophy or maximal strength
- Key modulator: separating endurance and strength by 3+ hours eliminates most interference within same-day sessions
- For recreational athletes (not elite), interference is minimal when total weekly volume is managed

## 2. Climbing + Upper Body Hypertrophy Interference

- Climbing is a hybrid stimulus: strength-endurance for forearms/fingers, strength for back/shoulders
- Primary interference risk: local muscular fatigue in pulling muscles (lats, biceps, forearms), not systemic
- Climbing does NOT meaningfully interfere with pushing muscles (chest, triceps, front delts)
- Back/pull hypertrophy work should be spaced 48-72 hours from hard climbing sessions
- Pushing work (bench, OHP) can be done the day after climbing with minimal interference

## 3. Sequencing Recommendations

- Climb Monday -> Push (chest/shoulders/triceps) Tuesday = fine, different muscle groups
- Climb Monday -> Pull (back/biceps) Tuesday = not ideal, same muscle groups under-recovered
- Ideal: place pull/back hypertrophy work 48+ hours after hard climbing
- Low-intensity climbing (easy routes, technique work) has much lower recovery demands

## 4. Auto-Regulation (RPE/RIR) for Variable Equipment

- RPE (Rate of Perceived Exertion): 1-10 scale; RPE 8 = could do 2 more reps
- RIR (Reps in Reserve): directly states remaining reps; RIR 2 = 2 reps left in tank
- For hypertrophy: work at RIR 0-3 (RPE 7-10)
- For strength: work at RPE 7-9 with lower rep ranges
- RPE/RIR solves the variable equipment problem: progressive overload is tracked by effort level, not absolute load
- Same movement pattern at different gyms: as long as RPE target is hit, adaptation stimulus is equivalent
- Progression method: keep same RPE target, accumulate reps over weeks, then increase load when rep target exceeded

## 5. Movement-Pattern-Based Programming

Best approach for varying equipment availability:

| Pattern | Examples (interchangeable) |
|---------|---------------------------|
| Horizontal Push | Bench press, DB press, push-ups, machine chest press |
| Vertical Push | OHP, DB shoulder press, landmine press, machine shoulder press |
| Horizontal Pull | Barbell row, DB row, cable row, machine row |
| Vertical Pull | Pull-ups, lat pulldown, cable pulldown |
| Squat | Back squat, goblet squat, leg press, Bulgarian split squat |
| Hinge | Deadlift, RDL, KB swing, hip thrust |
| Carry/Core | Farmer walks, suitcase carry, pallof press, plank variations |

- Program by pattern + RPE target, not specific exercises
- Swap exercises freely based on available equipment
- Track volume per pattern per week, not per exercise

## 6. Moderate Cardio (Badminton, Sustained Activity) Impact

- Moderate-intensity cardio 2-3x/week does NOT meaningfully impair hypertrophy
- Badminton is intermittent (short bursts + recovery) - closer to HIIT profile but moderate overall
- Key: separate from resistance training by 3+ hours if same day, or do on different days
- Adequate caloric intake is the critical variable - cardio only hurts gains when in caloric deficit
- Horse riding (~1hr moderate) has minimal interference with strength adaptations

## 7. Practical Weekly Template Principles

For a climb/strength/multi-sport athlete:
- Hard climbing days: treat as pull day from recovery perspective
- Day after climbing: push-dominant or lower body work
- Space pull/back hypertrophy 48+ hours from hard climbing
- Use RPE 7-8 as default training intensity for hypertrophy sets
- 1 rest day minimum between consecutive hard sessions on same muscle groups
- Moderate cardio (badminton, riding) can go on any day without interference concern
