## 案例数据

- 对象：24 岁男性，29.1% 体脂（DEXA 验证），引体向上达不到初一及格线
- 90 天结果：体脂降至 12%，净肌肉量有所增加，内脏脂肪下降，疾病风险持续降低
- 使用 DEXA（双能 X 射线）验证，误差约 1-2%

## 阶段策略

**前两周（习惯建立期）：**
- 只要求三件事：起床后量体重、自拍、下班后 15 分钟有氧
- 目的：不急于上强度，先建立运动习惯，观察是否能坚持
- 不控制饮食，前两周体重仅减少 1.6 kg

**第 17 天起：加入饮食控制**
- 提供具体饮食和训练计划
- 发现新手对食物营养完全没有概念（主食：面包、牛肉面、螺蛳粉、方便面，缺乏蔬菜）
- 控制饮食是真正的考验，也是了解营养知识的过程

**中后期：力量训练 + 逐步增加有氧**
- 一周五天力量训练，用 app 记录每天训练容量，防止肌肉流失
- 有氧时长逐渐增加
- 加入饭后散步 10-20 分钟（帮助消化 + 无意识增加消耗）
- 后期出现极度疲劳感（是正常阶段现象）

## 成功减脂的五个核心要素

1. 少吃多动（热量亏损）
2. 摄入足够蛋白质
3. 加入力量训练（在减脂的同时保留甚至增加肌肉量）
4. 坚持足够长的时间
5. 同时做到以上四点

## 习惯坚持的心理机制

- 前 30 天靠外界压力（他人监督、拍摄视频）坚持
- 后期不再需要监督，因为他开始享受到健身带来的好处并喜欢这个过程
- 建议新手：一开始主动寻找外界压力（教练或互相监督的伙伴），先逼自己坚持一段时间，在过程中找到乐趣

## 减脂期常见的7个错误

**错误1：把高糖早餐当"健康餐"**
- 水果、橙汁、牛奶麦片含大量糖分，显著提高胰岛素水平
- 胰岛素升高不仅阻止燃脂，还会促进脂肪囤积
- 减脂期碳水应选择胰岛素波动较小的复合型碳水（燕麦、糙米），而非精制糖和水果汁

**错误2：喝高热量咖啡**
- 拿铁、卡布奇诺含大量奶和糖，热量可达400-500卡
- 美式咖啡仅约70卡，是减脂期理想选择

**错误3：过度依赖减脂补剂**
- 左旋肉碱、燃脂丸只是辅助刺激物，不能替代饮食和训练
- 新手应先掌握基本饮食原则，再考虑补剂

**错误4：只靠跑步减脂**
- 跑步对体重偏重人群膝盖冲击大；替代方案：椭圆机、登山机、爬楼梯更安全

**错误5：恐惧脂肪**
- 摄入脂肪不直接导致变胖，关键是总摄入量是否超过消耗量
- 健康脂肪（牛油果、坚果的不饱和脂肪）对心血管有益，减脂期可适量摄入
- 需避免的是反式脂肪（油炸食品、汉堡中的加工脂肪）

**错误6：出汗多≠减脂多**
- 汗蒸/桑拿后体重下降是水分流失，不是脂肪消耗
- 有氧运动中应监控心率，而非汗量

**错误7：频繁测量体重**
- 一天测多次无意义，体重波动主要反映水分变化
- 正确方法：每天可测，但以周为单位比较趋势

## 减肥方法评估与真相（8人访谈）

### 各类方法评估

**节食（极端减少热量）**
- 严重降低血清素水平 → 易怒、焦虑、抑郁
- 负面情绪积累 → 暴饮暴食 → 恶性循环
- 停止节食后体重反弹，比原来更胖

**减肥药**
- 正当用途：病理性肥胖的医学干预手段
- 普通人自行吃减肥药的风险：头晕、暴饮暴食、心悸等

**中医减肥（针灸/拔罐）**
- 效果有限（2012年文献：117位超重者，食用中药3个月后减了1.5kg，未控制饮食）

**袖状胃切除术（切胃手术）**
- 将大部分胃切掉，饥饿荷尔蒙减少，不容易饿 + 饱得快
- 严重副作用：长时间胃食道反流；手术后无法用食物缓解焦虑 → 抑郁倾向
- **不推荐**因外貌焦虑考虑此手术

**循序渐进减肥法（推荐）**
- 每次只给自己增加一点点难度，身体和心理都更容易维持
- 示例：第一阶段不吃晚饭 → 加入运动 → 控制午餐 → 精细饮食+力量训练

### 食物上瘾机制（节食导致暴饮暴食的根源）

- 最容易上瘾的三种成分：**反式脂肪 + 咖啡因 + 糖**
- 三者混合（如奶茶）→ 产生更多多巴胺 → 更容易上瘾
- 节食会导致血清素下降 → 情绪问题 → 暴饮暴食 → 恶性循环

### 关于肥胖的认知
- 大体重者并非都是懒惰或缺乏自制力
- 生长发育期大量摄入垃圾食品 → 永久性影响荷尔蒙水平（瘦素、胰岛素）→ 反过来影响行为
- 每个人基因、环境、脂肪细胞数量不同，不应用自己的经历评判他人
