## 糖的生物学机制

### 为什么人类天生喜欢甜味
- 远古时期水果/蜂蜜 = 稀缺的能量来源，进化出用味觉识别甜味的功能
- 大脑逻辑："明天可能吃不到，今天多吃一些"
- 成千上万年后大脑没有进化太多，但环境已发生巨大变化 → 不有意识对抗本能就会过量摄入糖

### 糖 vs 碳水的区别

| | 糖（蔗糖/双糖） | 复合碳水（多糖） |
|---|---|---|
| 结构 | 双糖，直接分解成两种单糖 | 多糖，需多步转化为葡萄糖 |
| 吸收速度 | 极快 | 缓慢 |
| 血糖影响 | 迅速升高后出现低血糖 | 平稳升高 |
| 低血糖后果 | 疲劳、焦虑、犯困、注意力不集中、想吃更多碳水 | 不明显 |

### 糖与皮肤的关系
- 糖摄入后血液中糖与蛋白质/脂肪结合 → 产生**糖基化终产物（AGEs）**
- AGEs破坏胶原蛋白 → 皮肤变脆弱、加速衰老
- 多项研究发现：低糖饮食会缓解长痘（无直接因果证明，但相关性强）

## 30天断糖实验（极限版）

**戒断内容**：
- 添加糖（奶茶、冰激凌、可乐等）
- 含天然糖的水果（苹果、香蕉、西瓜、胡萝卜等）
- 牛奶、酸奶、新式调料
- 代糖/无糖零食（含甜味剂的食物）

**实验前的隐藏糖分（用方糖展示）**：
- 一块巧克力饼干：9g糖（约3颗方糖）
- 蛋白棒：9g糖
- 酸奶：29g糖
- 牛奶：12g乳糖
- 果汁：23g果糖
- 老干妈/醋/酱油中也含糖

## 主要结论

**体检数据变化（30天后）：**
- 血糖水平稳定（未见显著变化）
- LDL（不好的胆固醇）有一定程度下降
- 睾酮：仍然偏低（与断糖关系不明确）
- 体重和体脂：有一定程度改善
- 皮肤：痘痘情况好转

**代糖/甜味剂问题**：
- 断掉代糖/甜味剂一段时间后，味觉会变得更敏感
- 以前觉得无味的食物开始觉得好吃（类似"多巴胺戒断"效果）

## 作者不推荐极限版断糖的原因

- 几乎每天必须自己做饭，极其耗时
- 社交场合（朋友生日、火锅聚餐）极难维持
- 长期不现实，容易反弹

## 实用控糖策略（渐进法）

按难易程度，每个阶段坚持到适应再进阶：
1. 有糖饮料 → 无糖饮料
2. 一周内不喝任何带甜味的饮料
3. 连续3天断掉所有带添加糖的甜食
4. 连续7天断掉所有带甜味的食物（包括水果和调料）

**周期：如果每3个月戒断一次，可以极大缓解对甜食的渴望**

**核心原则**：目标不是戒断，而是学会控制。辨别哪些食物含糖、如何处理和它的关系，是需要掌握的技能。
