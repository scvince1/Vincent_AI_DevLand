## 熬夜与衰老的科学

- 2022年文献：长期倒夜班者，每天不同时段荷尔蒙分泌水平不同且峰值降低 → 大幅增加各种疾病风险
- 2024年文献：即使长期倒夜班者有更多深度睡眠，深度睡眠质量也会明显下降 → 加速大脑衰老

**睡眠不规律的连锁效应：**
- 每天靠咖啡维持精神
- 运动后恢复不足、更容易受伤
- 睾酮水平下降（作者亲测：2022年535 → 当年489）

## 决定睡眠质量的四个关键因素（Matthew Walker《Why We Sleep》）

1. **规律性（Regularity）** — 最重要
2. **连续性（Continuity）** — 不中断
3. **时长（Duration）** — 7-9小时
4. **深度（Depth）** — 深度睡眠比例

四者相互关联，缺一都会损害睡眠质量。

## 改善睡眠的6个方法

### 1. 定期戒断或减少咖啡因
- 如果一直有咖啡因在体内：降低深度睡眠、推迟眼动睡眠（REM）、减少总睡眠时长
- 建议：定期戒断咖啡因，或换成低因咖啡

### 2. 起床后立刻接触强光
- 起床后接触日光 → 提高早晨皮质醇水平 → 晚上更好产生褪黑素
- **注意：** 不抹防晒直接出门有紫外线老化皮肤风险 → 可用1万照度以上的摄影灯代替
- 即使在室内，靠近窗户或有强自然光也要抹防晒

### 3. 观察睡前静息心率
- 静息心率可直观反映神经和代谢系统是否进入准备入睡的状态
- **让睡前心率升高的因素（应避免）：** 酒精、咖啡因、过晚的晚餐、焦虑情绪
- 心率高于正常水平时，可通过冥想、泡澡等主动降低

### 4. 检查睡眠呼吸暂停
- 研究发现全球约10亿人有此问题，是影响睡眠连续性的主要原因之一
- 轻度打鼾的改善措施：换用较高枕头、保持侧睡、使用鼻孔扩张器

### 5. 多样化训练类型
- 力量训练 + 有氧之外，还需要融入：
  - **平衡训练**（随年龄增长平衡能力急剧下降）→ 多选择单边自由重量动作
  - **最大摄氧量训练**（VO₂max 随年龄快速下降）→ 一周一次HIIT冲刺跑
- 来源：《Outlive》书中对长寿目标的训练建议

### 6. 建立固定睡前流程
- 生活充满不确定性，固定流程可将睡觉变成可复制的条件反射
- 通过固定流程最大化应对不确定性的干扰

## 关于褪黑素

- **避免长期服用褪黑素：**
  - 研究：仓鼠连续服用5周后多巴胺水平下降约50%
  - 作为激素，长期使用会打乱昼夜节律、抑制多巴胺水平

## 理想作息建议（Bryan Johnson模式）

- 晚11:00入睡 → 睡满8小时 → 早7:00起床 → 两小时（早餐 + 运动）→ 9:00开始工作
- 七天保持一致，关键是**规律性**

## 实验结论

- 完全规律作息约1周后：睡眠质量明显改善，不需要咖啡因，到点自然有困意
- 戒断咖啡因的前4天会有头疼和犯困，第4天后能量稳定
- 核心原则：不必追求精确到分钟，但要持续了解自己、实践改变

## 昼夜节律基础（来自2022年内容）

### 人体内置时钟
- **人体内置时钟是25小时，而非24小时**（1972年法国科学家Michel Siffre洞穴实验）
- 这解释了为什么隔离期间人们会越睡越晚：身体遵循大于24小时的内在节律

### 调节时钟的核心信号：日光

**视交叉上核（Suprachiasmatic Nucleus, SCN）**：光透过眼睛到达该区域，根据光的强弱判断当下时间，控制荷尔蒙的释放

**为什么室内灯光不够用**：即使阴天，室外光照度也远高于室内最强灯光

### 正常昼夜节律下的荷尔蒙时间表

| 时间 | 荷尔蒙/状态 |
|---|---|
| 早晨6点 | 皮质醇开始升高（掌管压力/清醒） |
| 7-9点 | 睾酮达到峰值 |
| 9-11点 | 警觉性最高，逻辑思考能力最佳 |
| 下午2-4点 | 血清素分泌（稳定情绪） |
| 下午5点 | 肌肉力量达到峰值 |
| 晚上7点后 | 皮质醇下降，褪黑素开始升高 |
| 晚上10点 | 肠道蠕动变慢 |
| 凌晨12-2点 | 生长激素产生（帮助长高、增肌减脂） |
| 凌晨0-4点 | 存储记忆的深度睡眠 |

### 熬夜的研究数据
- 2002年（8000+工作者）：轮夜班工作者比上日班者更容易得流感和肠胃炎
- **2007年世卫组织将轮夜班归为致癌物**
- 男性睡眠不足5小时，8天内睾酮下降10-15%（2011年研究）
- 睡眠不足5小时的男性睾丸明显小于睡7小时以上者

### 三个调整昼夜节律的方法（2022年视角）

**方法1：利用日光调节**
- 早晨：起床后第一时间去室外感受日光
- 傍晚：感受日落时光（由亮变暗的过程）→ 触发褪黑素分泌
- 睡前3小时：手机和电脑降低亮度，调整夜间模式

**方法2：12小时间歇性断食**
- 起床后1小时内开始第一餐，睡前3小时停止进食 → 形成12小时空腹期
- 帮助肠胃提前预判消化时间，避免因肚子不适而影响入睡

**方法3：规律时间运动**
- 固定时间（推荐早晨）+ 结合接触日光效果最佳
- 晚饭后快走20分钟也有效，若能赶上日落更好

### 关键原则

- **没有统一正确的作息时间**：每个人根据工作和生活情况找到能长期坚持的规律作息
- 睡够时间 + 规律 > 睡得早但不规律
- 推荐每天睡7-9小时，一周至少1-2天睡到自然醒

## 睡眠对增肌的优先级（来自2019年内容）

**增肌三要素的正确优先顺序：**
1. **睡眠**（恢复 + HGH分泌 + 肌肉修复）
2. **饮食**（足够热量和蛋白质）
3. **训练**（提供刺激）

**睡眠紊乱（昼夜节律失调）的连锁影响：**
- 生物钟混乱 → 产生强烈厌食情绪 → 热量严重不足 → 身体无法合成肌肉，甚至分解肌肉
- 睡眠长期不规律 → 加速皮肤老化（皱纹增多、黑眼圈）

**增肌中的睡眠机制：**
- 训练后摄入足够蛋白质，在睡眠中身体产生HGH（人类生长激素），加快新陈代谢，帮助减脂增肌
- 非快速动眼睡眠（深度睡眠）中，流入肌肉的血液增加，加快肌纤维修复
- 睡眠不足时，即使体重下降，**燃脂量只有正常的1/2**

**实践意义：**
- 调整生物钟应该优先于任何健身目标
- 如果睡眠和饮食不规律，暂停其他非必要事项，先把作息调整过来
