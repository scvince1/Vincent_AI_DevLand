## 不同运动的卡路里消耗（以82kg成年男性为例）

| 运动 | 每分钟消耗 |
|------|----------|
| 什么都不做 | 1.4卡 |
| 仰卧起坐 | 4卡 |
| 大部分力量训练 | 5卡 |
| 深蹲 | 7.2卡 |
| 椭圆机 | 7.8卡 |
| 中等强度自行车 | 9.8卡 |
| 慢跑 | 10卡 |
| 打篮球 | 11.4卡 |
| 跳绳 | 17.7卡 |

数据来源：Compendium of Physical Activities（身体活动汇编，2001年）

## 关键误区：单位时间卡路里最高≠最减脂

**原因1：可持续时间不同**
- 跳绳 17.7卡/分钟 × 15分钟 = 266卡（很多人根本坚持不了15分钟）
- 中等强度自行车 9.8卡/分钟 × 30分钟 = 294卡（大多数人可轻松完成）

**原因2：心率越高，供能来源改变**
- 心率达到最大心率80-90%时，主要供能来源是肌糖原，而非脂肪
- 中低强度运动（心率在最大心率60-70%）时，脂肪作为供能来源的比例反而更高

## 减脂心率区间

**脂肪燃烧最优区间：**
- 公式：(220 - 年龄) × 60% ~ 70%
- 例：30岁 → 目标心率 114~133 bpm
- 在此区间长时间维持，脂肪利用率最高

## HIIT的真相

- HIIT理论上效率极高（运动后持续消耗热量，即EPOC效应）
- 但真正意义上的HIIT要求心率持续在最大心率85%以上，每组都近似力竭
- 现实问题：绝大多数人无法达到真正HIIT的强度（心理上拒绝达到那种不舒适状态）
- 新手还有受伤风险
- 结论：HIIT从理论上高效，但并不适合所有人

## 推荐的减脂训练方案

**两个核心原则：**
1. 无论做什么运动，将心率维持在减脂区间（60-70%最大心率），并配合心率监测设备
2. 融入力量训练

**2012年研究：** 对比只做有氧、只做力量、力量+有氧三组——
- 力量+有氧组体重与其他两组无太大差异
- 但在**体脂率、肌肉量、整体体型改变**上产生最大变化

**推荐训练结构（用于减脂）：**
- 复合动作 + 力量训练（多肌群参与，消耗卡路里更多）
- 组间用低强度有氧动作（跳绳、开合跳、高抬腿）维持心率
- 保持全程心率在减脂区间

**示例动作组合：**
- 弓箭步+肩膀推举（腿部+核心+肩膀）
- 俯身划船+硬拉（背部链条）
- 俯卧撑+登山者（胸部+核心）
- 相扑硬拉+二头弯举（大腿内侧+臀+手臂）
- 单臂推举+起身（胸部+核心稳定）

> 注意：如果主要目标是增肌，仍推荐传统先做力量后有氧的方式。

## HIIT vs 有氧：5个维度对比

### 1. 门槛难度
- **有氧运动**：门槛低，任何水平都可以开始（跑步、快走、登山机、划船机等）
- **HIIT**：门槛高，理想HIIT要求每组达到个人极限；无运动基础者有受伤风险

### 2. 卡路里消耗效率
- **HIIT更高**：运动中 + 运动后持续消耗（后燃效应）
- **有氧**：主要在运动过程中消耗，结束后效果有限
- 注意：消耗卡路里效率高 ≠ 燃脂效果更好

### 3. 心肺功能
- 两者均有益于心肺，但HIIT心率要求更高，心肺提升也更大
- 实测体验：做HIIT后，相同重量的力量训练能多做1-2个

### 4. 对力量训练的干扰
- **HIIT干扰力量训练**：HIIT和无氧训练同用糖原供能系统，HIIT后做力量表现下降；力量后做HIIT也下降
- **有氧不干扰力量训练**：有氧主要使用有氧供能系统，这也是推荐力量训练后再做有氧的原因

### 5. 减脂效果
- **有氧运动更适合备赛级减脂**：精确可调（相同强度+增加时间=精确增加消耗）
- HIIT消耗量难以精确监控，备赛运动员几乎全用有氧
- **2021年综合文献（54篇对比研究）**：对于长期减脂，有氧和HIIT没有明显差异，优先考虑时间和可持续性

## 人群选择建议

**推荐有氧运动的人群：** 初学者、需要精确控制减脂进度的人、同时进行力量训练的人、备赛人群

**推荐HIIT的人群：** 有一定运动基础、时间有限、希望提升心肺功能的人

**最优方案：** 两种结合，HIIT目标心率 = (220 - 年龄) × 85%-95%，每周3次，与有氧穿插进行

## HIIT 4种方案

### 方案1：高抬腿间歇冲刺（徒手，全身）
- 热身：30秒开合跳 × 2组（组间休息30秒）
- 20秒**全力**高抬腿冲刺 → 休息40秒 → 重复，总时长10分钟
- 动作要点：膝盖抬到与地面平行，换腿时脚尖先着地
- 体重较大或膝盖不好：改用后踢腿

### 方案2：平板支撑+静力蹲改良（新手/大体重友好）
- **平板支撑跳腿**：保持平板支撑，用腹部爆发力将双腿收回再打开；20秒 → 休息40秒
- **静力蹲摸脚尖**：保持深蹲姿势，用一侧手摸另一侧脚尖，换边；20秒 → 休息40秒
- 相对安全，适合刚开始减脂的人

### 方案3：哑铃梯子动作（中等难度，家/健身房均可）
- **哑铃摆铃（Romanian Deadlift变式）**：双脚打开，用臀部爆发力将哑铃顶起，接二头弯举；30秒 → 休息60秒
- **哑铃深蹲推举**：深蹲至大腿平行地面，起身时爆发力推举；30秒 → 休息60秒

### 方案4：有氧器械HIIT（最推荐，对关节友好）
- **自行车（首选）**：无速度上限 + 不冲击膝盖；热身2分钟 → 30秒全力冲刺 → 休息30秒 → 重复
- **划船机**：额外动用上肢力量，可与自行车交替
- **跑步机**：不推荐（速度上限低，对膝盖有冲击）

## 备赛级有氧实操经验

### 为何选择登山机和跑步机爬坡
- 登山机、跑步机爬坡：机器带着走，强制保持固定速度，心率稳定在燃脂区间
- 椭圆机、自行车：需主动发力，疲劳时人会不自觉降速，心率难以稳定

### 稳态有氧 vs HIIT（备赛视角）
- 稳态有氧可精确计算消耗（如今天消耗300卡），便于根据效果调整
- 体重停滞时，备赛教练通常调整稳态有氧时间，而非增加HIIT（无法精确监控）

### 空腹有氧（个人经验，学术界尚有争议）
- 潜在优势：同等强度下减脂效果更明显
- **不推荐每天空腹有氧**：有氧时睾酮水平下降，早起空腹有氧会拉低全天荷尔蒙水平
- Jeremy的替代方案（备赛选手）：早晨空腹**低强度快走**1小时（不影响荷尔蒙）

### 高频有氧注意事项（备赛级别）
- 过多有氧 → 皮质醇升高 → 反而难以减脂
- 应定期进行恢复日，或换有氧形式
- 理想的有氧应该是**你喜欢或享受的形式**
