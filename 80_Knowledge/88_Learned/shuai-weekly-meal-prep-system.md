## 核心思路
参考健康快餐厅（Sweetgreen/Chipotle）模式：预先批量制备各类食材 → 随时组合成不同口味一餐。好处：保持新鲜、避免每天吃一样、节省时间。

**目标：** 约90分钟内完成4-5天的所有食材备制

## 食材选择原则
- 每类食材做好后冷热皆可食，任意组合都好吃
- 蛋白质 + 蔬菜 + 碳水 = 完整健康餐

### 推荐食材清单
**蛋白质：** 鸡胸肉、虾、三文鱼  
**蔬菜：** 西兰花、羽衣甘蓝、胡萝卜、小卷心菜  
**碳水：** 小土豆、红薯、白米+黑米混合  

## 制备流程（约90分钟）

### 蛋白质（约20分钟，烤箱批量）
- 鸡胸肉：切块，吸干水，加盐、黑胡椒、牛油果油，220°C 烤20分钟
- 三文鱼和虾：同样处理，分放不同烤盘，同时烤制
- 效果：外焦里嫩；无烤箱可用空气炸锅或煎制

### 蔬菜（约20分钟，水煮）
- 西兰花：切块，沸水煮1分钟（保留脆感），捞出马上沥干
- 胡萝卜：切片，煮2.5分钟
- 羽衣甘蓝：去根茎，煮60秒
- 小卷心菜：对半切，煮2.5分钟
- 保存技巧：保鲜盒底垫厨房纸防吸水变软；真空保鲜盒可存4-5天

### 碳水（与蛋白质同时进行）
- 白米+黑米：1:1:2（米:米:水），电饭锅30分钟
- 小土豆：对半切，切面朝下，245°C 烤20分钟（至表面略焦）
- 红薯：去皮切块，加橄榄油和盐，同温度烤20分钟

**存储：** 所有食材放凉后分装冷藏

## 组合示例

| 组合 | 食材 | 特点 |
|------|------|------|
| 香辣彩虹鸡胸饭 | 米饭+胡萝卜+卷心菜+羽衣甘蓝+鸡胸+剁椒酱 | 50g蛋白，6.7g脂肪，40g碳水 |
| 墨西哥风味蔬菜碗 | 鸡胸+西兰花+小土豆+墨西哥辣酱 | 仅347kcal |
| 烧烤虾 | 虾+羽衣甘蓝+零糖烧烤酱 | 低卡替换方案 |
| 增肌三文鱼碗 | 三文鱼+羽衣甘蓝+红薯+牛油果+坚果+米饭 | 44g蛋白，32g脂肪，~100g碳水 |
| 韩式健康餐（生酮友好） | 三文鱼+蔬菜沙拉+牛油果+海苔碎+芝麻+辣酱 | 适合第一餐，不升血糖 |

## 关键技巧
- 根据目标（减脂/增肌）调整碳水和脂肪来源的量，蛋白质来源可以互换
- 高纤维碳水（红薯、黑米、土豆）比白米有更强饱腹感，同量摄入下更有助减脂
- 各类调味酱（剁椒、辣酱、照烧酱）是让备餐不单调的关键
- 用零脂肪喷油代替直接倒油，可大幅减少额外脂肪摄入
- 低盐酱油、零卡辣酱、果醋（助消化、防胀气、低卡）是减脂期好调料

## 扩展食谱库

### 蛋白质主食类

**手撕鸡胸肉沙拉**
- 水煮鸡胸时加半个柠檬去腥
- 搭配洋葱、青椒、小葱，调味用低盐酱油+果醋
- 营养：54g蛋白，3g脂肪，4g碳水，仅276kcal
- 适合晚上加餐（几乎无碳水）

**黄油迷迭香牛排**
- 牛排两面撒盐、黑胡椒、孜然；锅面抹少量黄油防粘
- 正反各煎2分钟，调小火加黄油和迷迭香，用勺子不断浇汁
- 三分熟约2-3分钟；七分/全熟可额外入烤箱5分钟

**土豆芦笋烤鳕鱼**
- 烤箱：先烤土豆15分钟，再加芦笋、番茄、鳕鱼烤20分钟（220°C）
- 加蒜粉、盐、黑胡椒调味，每份挤柠檬汁
- 冷藏可保存3天，适合批量备餐

### 早餐/便携类

**蛋白鸡蛋小松饼（烤制，无面粉）**
- 食材：鸡蛋、1/4杯无糖牛奶、菠菜、小西红柿、熟鸡胸肉
- 烤箱175°C，25分钟；每两个：23g蛋白，11g脂肪，1g碳水，195kcal
- 可批量制作冷藏，作早餐或便携零食

**蘑菇菠菜鸡蛋卷（减脂早餐）**
- 锅内喷油，炒蘑菇（加盐、胡椒），加菠菜，打入鸡蛋，煎熟后卷起
- 营养：18g蛋白，7g碳水，15g优质脂肪；准备时间：5分钟

**蓝莓蛋白燕麦（练前一餐）**
- 燕麦+水+切片蛋白棒+蓝莓，冷藏8小时（隔夜燕麦）
- 营养：约20g蛋白，71g碳水，7g脂肪；训练前1小时食用

### 沙拉/极简类

**鸡蛋菠菜沙拉（可含牛油果）**
- 食材：鸡蛋、牛油果（可选）、小西红柿、洋葱、香菜、菠菜
- 含牛油果版：约840kcal，47g蛋白，58g脂肪，33g碳水
- 去牛油果去蛋黄版：仅178kcal，28g蛋白，1g脂肪，14g碳水（极佳减脂餐）

**吞拿鱼/沙丁鱼罐头沙拉（极简晚餐）**
- 深绿色混合沙拉 + 水浸金枪鱼或沙丁鱼罐头（选**水浸**非油浸）
- 加醋调味，可加2颗杏仁；营养：约17g蛋白，4g碳水，2g脂肪
- 准备时间：2分钟；强烈食欲时可作加餐

### 快手主食类

**鸡胸肉+西兰花+米饭（经典减脂组合）**
- 鸡胸肉切片 + 零脂肪喷油；西兰花沸水煮20-30秒（保留脆感），凉水冲洗
- 即食米饭微波炉2分钟；可加零卡辣椒酱调味
- 准备时间约8分钟；碳水较多，适合**训练后**食用

**罗非鱼+芦笋+红薯**
- 罗非鱼解冻后小火慢煎；芦笋水煮法；红薯微波炉加热（或水煮）
- 相比鸡胸+西兰花更易消化，红薯饱腹感更强；适合减脂后期/备赛期

**鸡肉沙拉三明治（无需开火）**
- 全麦面包（选低糖少添加剂）+ 火鸡肉片 + 牛油果（可选）+ 蔬菜
- 准备时间3分钟；适合宿舍/出差场景

**鸡胸番茄意面**
- 100g意面（无糖番茄酱），煮面时加少量橄榄油防粘
- 番茄酱量控制以避免碳水超标；翻面至金黄后加入番茄酱和煮面汤收汁
