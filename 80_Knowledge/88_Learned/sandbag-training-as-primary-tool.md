# Sandbag Training as Primary Tool

## Best Exercises (for carry/hold/toss 60kg person goal)
**Carry variations:** Bear hug carry, shoulder carry (L/R), Zercher carry, bear hug walk + squat combo
**Strength builders:** Bear hug squat, ground-to-shoulder clean, sandbag-over-shoulder toss, shouldering to platform, sandbag get-up

## Sandbag vs Barbell for Carry Strength
- Sandbag SUPERIOR for functional carry: shifting load forces constant grip adjustment and bracing
- Greater core and stabilizer activation vs barbell equivalents (Luther 2015)
- Barbell builds raw maximal strength more efficiently
- Complementary: barbells for peak force, sandbags for real-world application

## Hypertrophy Potential
- Can drive meaningful hypertrophy if hitting 8-15 hard sets per muscle group near failure
- Effective for: upper back, traps, shoulders, core, legs
- Not optimal for isolation (e.g., bicep peak)
- Pair with dumbbells for isolation work

## Progressive Overload with Fixed 45kg
| Method | Example |
|--------|---------|
| Volume | 3×5 → 5×5 → 5×8 |
| Density | Same work, shorter rest |
| Tempo | 3-sec eccentric squats, paused reps |
| Distance/Duration | Carry 20m → 30m → 50m |
| Complexity | Bear hug squat → squat + carry → squat + toss |
| Grip difficulty | Handles → bear hug (no handles) → odd positions |
| Unilateral | Shoulder carry (one side loaded) |

## TFCC Interaction
- More wrist-friendly than barbell for many movements
- **Favor:** Bear hug grip, shoulder carries, Zercher position (forearm cradle)
- **Caution:** Overhead press (wrist deviation), floor-to-shoulder cleans (impact on catch)
- **Avoid:** Forceful ulnar deviation or heavy wrist extension under load
- Relaxed grip + braces; bag shape distributes load across forearms

## Sample Session (~30-40 min)
| Block | Exercise | Sets × Reps | Rest |
|-------|----------|-------------|------|
| A | Ground-to-shoulder (alt. sides) | 4×5 each | 90s |
| B | Bear hug squat (3s eccentric) | 4×8 | 90s |
| C | Bear hug carry | 3×30m | 60s |
| D | Shoulder carry (each side) | 3×20m/side | 60s |
| E | Sandbag-over-shoulder toss | 3×6 | 90s |
Finisher: mace 360s/10-to-2s, 3×10 each direction
