## 5种低卡减脂食物

### 1. 魔芋

- **热量**：每200g仅12卡、6g碳水（对比普通米饭：258卡/200g）
- 可替代米饭、面条等主食
- 注意：魔芋零食通常糖和钠含量很高，避免过度使用

**食谱1：吞拿鱼魔芋拌面**
- 材料：魔芋面、吞拿鱼罐头、酱油（+葱花芝麻可选）
- 翻炒吞拿鱼 → 加魔芋面 → 少量酱油搅拌2分钟
- 热量：102卡 / 6g碳水 / 21g蛋白质 / 1g脂肪

**食谱2：鸡蛋虾仁炒魔芋饭**
- 材料：魔芋米、虾仁、青椒、3个鸡蛋、蒜、盐胡椒
- 热量（含蛋）：332卡 / 9g碳水 / 44g蛋白质 / 14g脂肪
- 热量（去蛋）：117卡 / 低卡高蛋白

### 2. 西葫芦

- **热量**：每200g仅34卡，95%是水分
- 用刨丝器刨成面条状代替主食
- 同样200卡的情况下，西葫芦面体积远大于普通面条

**食谱：柠檬虾仁炒面**
- 材料：西葫芦2根、虾仁、蒜、柠檬、香菜
- 热量：165卡 / 12g碳水 / 27g蛋白质 / 2g脂肪

### 3. 花菜

- 低卡、高膳食纤维，饱腹感强
- 花菜米：每200g约50卡、10g碳水（对比米饭大幅降低）

**食谱：咖喱鸡胸花菜饭**
- 材料：鸡胸1片、花菜米1碗、洋葱、蒜、咖喱粉
- 鸡胸盐胡椒腌10分钟 → 炒洋葱蒜 → 加鸡胸 → 加花菜米和咖喱粉翻炒2分钟
- 热量：182卡 / 5g碳水 / 27g蛋白质 / 3g脂肪

### 4. 芹菜

- **热量**：每100g仅14卡
- 富含不可溶性纤维，助排便
- 含三正丁基本肽：有消炎和防止肌肉痉挛的作用（备赛运动员的零食选择）
- 口感脆，减脂期可当水果吃

**食谱：低卡高蛋白奶昔**
- 材料：杏仁奶 + 香蕉1根 + 芹菜2根 + 菠菜1把 + 冰块
- 热量：342卡 / 36g碳水 / 24g蛋白质 / 4g脂肪
- 去掉香蕉可大幅降低碳水

### 5. 米饼

- 由米饭制成，无添加物，保留零食口感
- 每块约40卡、8g碳水、1g蛋白质
- **减脂期**：体积大，更容易产生饱腹感
- **增肌/增重期**：体积小，容易不知不觉吃很多（适合增卡路里）

## 核心策略

不需要彻底改变饮食计划，只需**替换掉一两个主要热量来源**（如把米饭换成魔芋米，面条换成西葫芦面），就会有显著效果。
