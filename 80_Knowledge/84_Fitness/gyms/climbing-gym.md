# 常去攀岩馆

## 设施
- 深蹲架/硬拉区常被占
- 丰富弹力带、各尺寸杠铃片、部分哑铃
- 优质地面、大引体区、垫子
- 大跳箱（做台阶激活，不跳）
- Vincent 自带弹力带

## 可执行模式
| Pattern | 可用动作 |
|---------|----------|
| H-Push | 杠铃片地板卧推、俯卧撑变体 |
| H-Pull | DB划船、弹力带 |
| V-Push | DB肩推 |
| V-Pull | 引体向上区(加重) |
| Hinge | 杠铃片RDL、弹力带 |
| Knee | Box pistol squat、plate goblet |
| Carry | 杠铃片/DB farmer carry |
| Rot | 弹力带Pallof、plate chop |
