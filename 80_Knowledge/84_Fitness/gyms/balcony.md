# 阳台
_Vincent 的主训练场地_

## 器材
| 器材 | 规格 | 备注 |
|------|------|------|
| 沙袋 | 45kg(可加至70kg) | 70kg难控制有砸脚风险，仅明确指定时使用 |
| 棒铃 | 15/20/25 lb ×2对 | 旋转稳定性训练；TFCC兼容；25lb戴护腕 |
| BDG | 1个 | 热身玩具，不计入训练 |
| 哑铃 | 12lb ×2 | 仅侧平举等适配动作；俯卧撑手撑用（中立腕！） |
| 指力板 | 绳索式 | 当前体重不可用(>70kg限制) |
| 瑜伽砖+垫子+地毯 | — | 地面训练友好 |
| 引体向上木架 | — | 技术上可用，一般不用 |
| 弹力带 | 新的在路上 | 旧的已坏 |

## 可执行模式
| Pattern | 可用动作 | 备注 |
|---------|----------|------|
| H-Push | 哑铃俯卧撑(中立腕!)、沙袋地板卧推 | 护腕必须 |
| H-Pull | 弹力带划船、沙袋弯划 | 弹力带到货后 |
| V-Push | 棒铃推举、DB站姿推举 | 12lb适配 |
| V-Pull | 引体向上杆、弹力带下拉 | 少用 |
| Hinge | 沙袋RDL、沙袋good morning | 主力器材 |
| Knee | 沙袋goblet squat、pistol进阶 | |
| Carry | 沙袋熊抱行走(45kg) | 主力 |
| Rot | 棒铃360/10-to-2、弹力带旋转 | |

## 心理标签
- 练胸首选（去健身房练胸摩擦大）
- 随时可用、无需换衣下楼
- 光脚训练友好
