# 训练约束与伤痛管理
_最后更新: 2026-04-09_

---

## 1. 伤痛约束（不可违反）

| 部位 | 约束 |
|------|------|
| TFCC 双侧（右重） | 全程佩戴护腕(必须!); 优先中立握; 禁反握绳索下拉; 俯卧撑必须中立腕(哑铃/把手), 禁平手撑地 |
| 肩 | 禁颈后下拉; 深蹲前必须肩背颈激活 |
| 膝/髋 | 旧伤已康复, 高负荷深屈曲谨慎; CARs先探ROM |
| 手腕承重 | 俯卧撑~30kg/手超过临床TFCC耐受(~16kg/手); 护腕=必须; 疼痛=立即停止; 状态不好→退阶跪姿 |

---

## 2. 腕部 Session 标签系统（WLT）

每次 session 打一个腕部负荷标签：

| 标签 | 定义 | 典型活动 |
|------|------|----------|
| `wrist_load: 0` | 腕部负荷可忽略 | 腿部训练、有氧、legless攀爬、骑马散步 |
| `wrist_load: 1` | 中等腕部负荷 | 拉类力量训练、骑马(>30min主动缰绳)、棒铃flow、沙袋carry |
| `wrist_load: 2` | 高腕部负荷 | loaded crimp/pinch、动态攀岩、极限boulder、悬挂训练 |

**7日滚动规则：**
- 7日内 `wrist_load` 总和 ≤ 5 分
- `wrist_load: 2` 后，下次 session 不得为 2（强制冷却）
- 7日内出现 2 次 `wrist_load: 2` → agent 提醒

**硬停止（独立运作）：**
- 腕部 NRS ≥ 5（任意一侧）→ 零手指负荷
- 单次 session 出现 sharp pain → 立即停止腕部负荷活动

> WLT 系数为估算起点，前8-12周根据 Vincent 实际疼痛数据校准。

---

## 3. TFCC 康复/强化训练（Prehab Protocol）

2-3次/周，作为 `wrist_load ≥ 1` session 的热身组件：

| 动作 | 参数 | 目的 |
|------|------|------|
| 偏心旋前/旋后（轻哑铃/锤握） | 3×10/侧，极慢离心 | TFCC 韧性建设 |
| Rice bucket work | 2-3min | 前臂深层小肌群激活 |
| 等长尺偏/桡偏固定 | 3×15s/方向 | 腕部稳定性 |
| 握力球挤压 | 2×20 | 一般腕部力量 |

> 试运行1-2周感受量。Vincent 反馈太多可精简至 rice bucket + 偏心旋前/旋后。

## 4. 攀岩握法规则（基于TFCC, 非A2 pulley）

| 握法 | 安全性 | 规则 |
|------|--------|------|
| Jug | 最安全 | 优先使用 |
| Open-hand (正向edge) | 安全 | 默认握法 |
| Crimp | 中等(pulley风险) | 控制单次session总量 |
| Gaston/Side-pull | 中等(尺偏) | 注意腕位 |
| **Sloper** | **最高风险(TFCC!)** | **避免** |

> "sloper比crimp安全"是A2 pulley的建议，不适用于TFCC。Sloper在91kg下是最危险的受力方向。

## 5. 设备偏好

| 偏好 | 说明 |
|------|------|
| 不频繁换哑铃 | 同一set内保持同一重量 |
| 绳索换片OK | 插拔方便 |
| 硬拉不换片 | 一个重量从头到尾 |
| 不用史密斯机 | 太孤立 |
| 不穿鞋 | 光脚/袜子/拖鞋 |
| 不喜欢划船机 | 脚蹬不适配 |

---

## 6. 左右平衡规则

- 左侧全身性弱于右侧
- 优先单侧动作（哑铃 > 杠铃）
- 单侧动作左先右后，右侧跟左侧次数
- 禁止：单独"只练左边"、额外追加左侧单独组
- Log 中记录"侧"列(双/左/右)
