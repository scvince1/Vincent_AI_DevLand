# 组间休息 / Rest Intervals
_建立: 2026-04-14 | 来源: 3 路 sonnet 研究合流_
_用途: fitness-coach 推荐组间 + 特殊情境（TFCC/攀岩/羽毛球同日/碎片睡眠）调整依据_

## 0. 硬速查表（默认状态：HRV 正常 + 睡眠足 + 非交叉训练日）

| 目标 / 动作类 | 组间 | 关键引用 |
|---|---|---|
| 最大力量（≥85% 1RM, 1-5RM） | **3-5 min** | ACSM 2009; NSCA PTQ |
| 肌肥大复合（squat/DL/bench/row, 6-12RM） | **2-3 min** | Schoenfeld 2016; Grgic 2017 |
| 肌肥大孤立 / 机器（6-12RM） | **60-120s** | Grgic & Schoenfeld 2018 |
| 肌耐力（15+RM） | **<60s** | ACSM 2009 |
| 爆发力 / Olympic / 弹射 | **3-5 min** | NSCA |

## 1. 核心原理：旧"短休增肌"范式已倒

- **Schoenfeld 2016 (JSCR)**：3 min 组间 vs 1 min，8 周训练后四头和肱三头肥大显著更多。机制：容量负荷（volume load）才是肥大的近端驱动，短休系统性压缩容量。
- **Grgic 2017 (JSCR) 系统综述**：训练者 long rest 更有利（容量保护）；新手两者等效。
- **Schoenfeld & Grgic 2024 (Frontiers 贝叶斯 meta)**：容量等同条件下，rest interval 本身影响小（SMD ≈ 0.17）。**关键**：大多数中阶训练者在短休下无法匹配容量，所以实战中 long rest 仍胜出。
- **de Salles 2009 (Sports Medicine)**：复合动作 1 min 休息比 3-5 min 的次数下降更陡。
- **口诀**：如果你短休时无法维持容量 → 容量损失就是代价。

## 2. 背日 / 拉类专门考量

**握力独立限速**：
- 前臂屈肌 PCr 完整恢复约 3 min（Hörst, trainingforclimbing.com）；50s 覆盖第一个关键恢复阶段。
- 划船/下拉使用助力带（strap）可多做 1-3 次/组 → 说明握力疲劳早于背阔。
- 90-120s 标准肥大组间对于握力依赖的拉类是**不够的**，尤其训练后段。

**实战调整**：
- 划船、引体、高位下拉：建议 **2-2.5 min**，后段或 RIR≤1 组加到 3 min
- 中立握/V 杆/对握绳索：对腕和握更友好，可短 15-20s
- 反握/宽握正手：握力负担最大，组间拉长 15-20%

## 3. TFCC / 腕部受限专门协议

- **轮轴/旋转负荷在场**：组间 **3-5 min**（Physiopedia / Hooper's Beta 共识；2024 Thieme 康复综述）
- 直接 RCT 证据**空白**：TFCC rest interval 长度 vs 再损伤率没有对照实验。指引来自通用肌腱康复 load-management 外推。
- **凌喵规则**：TFCC 激活期 → 所有拉/推动作组间基线 **+10-15%**；护腕全程；腕 NRS≥3 立停；反握绳索动态黑名单。
- 等长起步阶段（康复早期）：10s 等长 × 3×10/天，2 周无痛后才上自由重量（PMC10921179）。

## 4. 攀岩 + 力量同周期

- **48h 最低间隔**（Hörst / Lattice / PhysiVantage 一致实践共识）
- **<24h 交叉**：降低握力依赖的举铁容量；避免 strap 掩盖握力疲劳信号
- **<48h 交叉**：拉类走机器/绳索控制轨迹，避免 deadhang 式大重量拉
- Tenosynovitis 回溯研究（PMC9220062）：攀岩损伤主因是**次间恢复不足**，非单次过载
- 直接 RCT 证据**空白**：无同周期攀岩+举铁前臂恢复动力学控制试验，属于已知证据缺口

## 5. 机器 vs 自由重量 CNS 成本

- **2023 meta (PMC10426227, BMC Sports Sci)**：自由重量 CNS + 稳定肌需求显著更高，单组中枢疲劳更大
- **2024 Frontiers meta (PMC11349676)**：动作复杂度直接改变次间疲劳累积
- **实战**：机器肥大组间 90-120s 够；自由重量复合（squat/DL/bench）≥3 min；accommodating resistance（band/chain）疲劳低于等强度自由重量

## 6. 并行训练（羽毛球 / 网球 + 举铁 同日）

**顺序**：
- 默认 **力量先 → 球后**（保留 mTOR 合成窗，避开 AMPK 抑制；Wilson 2012 / PMC5407958）
- 球类技能质量优先 → 可球先力量后，接受 5-10% 肥大信号损失
- **同日总 AU ≤ 800**（Foster session-RPE AU = RPE × 分钟）

**示例**：羽毛球 RPE6 × 60min = 360 AU；举铁 RPE7 × 45min = 315 AU；合计 675 AU ✅

**同日组间调整**：
- CNS 相位 + 同日球类 → 基线 **+60-90s**
- 代谢相位（肥大）+ 同日球类 → 标准组间，顺序灵活

## 7. Readiness Gate × 组间（凌喵核心调整矩阵）

| 状态 | 组间调整 | 组数调整 | 其他 |
|---|---|---|---|
| HRV 正常 + 睡眠足 | 标准 | 标准 | 按相位 |
| HRV 压制（<7d SWC 下限） | **+50%** | **-1 组** | 禁止加重量（Vesterinen 2016） |
| 睡眠碎片（Vincent 高频触发） | **+30-40%** | 标准 | 复合动作前置；代谢优先于 CNS |
| 周 strain 高（Foster monotony >2.0） | **+20-30%** | 用"低日"降 AU，非均降 | 防止疲劳累积（Foster 1998） |
| CNS 相位 + 同日球类 | 基线 **+60-90s** | 标准 | 球类后置或间隔 ≥3h |
| 饮酒上日 ≥3 杯 | **+30%** | 容量上限 70% | 已在 _rules 酒精规则 |

**睡眠碎片硬数据**：
- 一夜总剥夺 → 皮质醇 +21%，睾酮 -24%，MPS -18%（Dattilo 2020, PMID 31469710）
- 单关节力量保留，多关节复合显著下降（Fullagar 2015, Dattilo 2020）
- 碎片睡眠 = 慢性部分版本，长期累积

## 8. 决策 Flow（fitness-coach 调用时按此顺序问）

1. **HRV 状态** → 若压制：+50% rest, -1 set, 不加重量
2. **睡眠**：过去 24h < 6h 或碎片 → +30-40% rest, 倾斜代谢池
3. **今日有无球类/有氧**：有 → CNS 相位 +60-90s；代谢相位标准
4. **本周 strain**：Foster monotony >2.0 或 WLT≥4 → +20-30% rest
5. **动作类型**：
   - 腕旋转负荷 → 3-5 min
   - 复合 + CNS → 3-5 min（力量）/ 2-3 min（肥大）
   - 机器 / 孤立 → 90-120s
   - 拉类握力依赖 → 基线 +15-20%
6. **TFCC 在场**：最终值再 +10-15%

## 9. 常见错误 / 凌喵黑名单

- ❌ "1 分钟短休练大肌肉" → 2016 后已证伪（容量未等同前提下）
- ❌ 用 strap 掩盖握力疲劳然后短休 → 短期多做 reps，长期前臂过载
- ❌ 攀岩 <48h + 重拉 + 短休 → 肌腱炎高危组合
- ❌ 睡眠碎片日继续按标准组间走重组合 → CNS 进一步挖坑
- ❌ 酒后日按标准节奏走 → Readiness 已下调仍正常训练 = 典型 monotony 上升

## 11. 已知证据空白（凌喵坦白）

- 握力作为"独立限速器"的分离性 RCT → 无
- 同周期攀岩+举铁前臂恢复动力学 → 无对照试验
- TFCC 组间时长 × 再损伤率 → 零 RCT，全部外推
- 机器 vs 自由重量"所需休息"匹配协议直比 → 稀薄
