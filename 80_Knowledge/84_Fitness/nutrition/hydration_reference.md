## 1. 每日水分目标

### IOM 基线推荐 (成年男性)
- **总水分 (食物+饮品)**: 3.7 L (125 oz)
- **饮品部分**: ~3.0 L (101 oz) — 食物贡献约20%

### Vincent 调整目标
| 因素 | 调整 | 依据 |
|------|------|------|
| 体重 ~91 kg (偏高于IOM参考) | +10% | 体表面积/代谢率 |
| 高钠摄入 (常2,000-4,000+ mg/d) | +15-20% | 肾脏排钠需要额外水分; 虽然稳态下肾脏可通过浓度调节, 但超过UL的钠负荷增加排泄需求 |
| 力量训练 (中等出汗) | +500 mL (17 oz) on training days | [GSSI](https://www.gssiweb.org/sports-science-exchange/article/sse-92-dietary-water-and-sodium-requirements-for-active-adults) |
| 咖啡因摄入 (习惯性) | 不额外调整 | 见下方§咖啡因与水分 |

**Vincent 每日饮品目标:**
- **休息日**: ~115 oz (~3.4 L)
- **训练日**: ~132 oz (~3.9 L)

> 来源: [IOM via Calculatorian](https://www.calculatorian.com/en/articles/health/daily-water-intake-iom) | [PubMed 35943601](https://pubmed.ncbi.nlm.nih.gov/35943601/)

## 2. 咖啡因与水分平衡

### 核心发现: 习惯性饮用者不脱水
- 每日2-3杯咖啡的习惯性饮用者, 咖啡的补水效果与水相当
- 280mg以内咖啡因: 习惯性状态不影响利尿反应
- 高剂量 (6 mg/kg, 即Vincent约546mg): 急性利尿效应, 摄入后2-3小时尿量增加
- 4杯以上/天: 可能出现咖啡因诱导利尿

> 来源: [PLOS One - PMC3886980](https://pmc.ncbi.nlm.nih.gov/articles/PMC3886980/) | [PMC5563313](https://pmc.ncbi.nlm.nih.gov/articles/PMC5563313/)

### 水分计入系数 (Hydration Credit)
基于研究, 对习惯性咖啡因消费者:

| 饮品类别 | 水分计入率 | 说明 |
|----------|----------|------|
| 纯水 | 100% | 基准 |
| 低咖啡因饮品 (<100mg/份) | 95% | Diet Coke, 单杯茶 |
| 中咖啡因饮品 (100-200mg/份) | 90% | 单pod咖啡 |
| 高咖啡因饮品 (>200mg/份) | 85% | 双pod咖啡, 浓缩 |
| 补剂饮品 (纤维水/肌酸水) | 100% | 无咖啡因 |
| 蛋白奶昔 | 95% | 低咖啡因, 高渗透压略降吸收 |
| 酒精饮品 | 0-50% | 视量级; ≥3杯=净脱水 |

## 3. 弱追踪模式 (Minimal Tracking Schema)

### 设计原则
- Vincent 不会填详细表格
- 颗粒度: 容器级别, 不是毫升级别
- 每日总结时自动汇总

### 快速记录格式
```
☕ 大杯×1  |  🥤 DC×2  |  💧 瓶×3  |  🧪 纤维水  |  🥛 奶昔
```

### 容器标准化

| 代号 | 描述 | 容量 (oz) | 水分计入 (oz) |
|------|------|----------|-------------|
| ☕ 大杯 | 67oz杯冲2pods+冰→~50oz | 50 | 45 (90%) |

| 🥤 DC | Diet Coke 12oz罐 | 12 | 11 (95%) |
| 💧 瓶 | 标准水瓶 (假设16.9oz) | 17 | 17 (100%) |

## 4. 钠负荷与水分需求

### 研究结论
- 稳态下, 肾脏通过调节尿液浓度适应钠摄入范围, 不一定需要等比增加水量
- 但超过UL (2,300mg) 的急性高钠负荷会刺激渴觉中枢, 增加即时饮水需求
- 高钠日 (>3,000mg): 建议额外饮水500mL (~17oz)
- 极高钠日 (>4,000mg, 如04-11): 建议额外饮水1L (~34oz) + 高钾食物促排

> 来源: [PubMed 28614828](https://pubmed.ncbi.nlm.nih.gov/28614828/) | [PMC8778661](https://pmc.ncbi.nlm.nih.gov/articles/PMC8778661/)

### 钠-水分联动规则 (Agent用)
| 当日钠摄入 | 水分目标调整 |
|-----------|-------------|
| < 2,300 mg | 标准目标 |
| 2,300-3,000 mg | +10% (~12 oz) |
| 3,000-4,000 mg | +15% (~17 oz) |
| > 4,000 mg | +25% (~30 oz) + 提醒高钾食物 |

## 5. Agent 决策规则

### 自动追踪行为
- Vincent 报告任何饮品时 → 同时更新水分追踪栏
- 日总结时 → 自动计算水分进度 vs 目标
- 联动 Na/K 监控: 高钠日自动上调水分目标

### 提醒触发
| 条件 | 动作 |
|------|------|
| 日水分计入 < 80% 目标 (截至最后记录) | ⚠️ "水分偏低, 考虑再喝一瓶水" |
| 咖啡因 > 400mg 且水分 < 100% 目标 | ⚠️ "高咖啡因日, 水分尤其重要" |
| 高钠日 + 水分 < 调整后目标 | ⚠️ 联合提醒钠+水 |
| 训练日未记录训后饮水 | 被动等待, 不主动问 (弱追踪原则) |

### 不做的事
- 不追问每一口水
- 不在水分充足时发提醒
- 不因单日不达标而发警告 (只在趋势性不足时提及)
