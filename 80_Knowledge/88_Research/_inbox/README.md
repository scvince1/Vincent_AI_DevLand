# _inbox/

PDF drop zone for paywall-fetched materials.

Vincent drops PDFs downloaded from UChicago library here. 凌喵 scans this folder periodically (or on command), moves files into the appropriate concept dir, and updates `_sources.md` + strikes through `_todo-paywall.md` rows.

Naming at drop time does not matter. 凌喵 will rename to `<author>-<year>-<short-title>.md` format during ingestion.
