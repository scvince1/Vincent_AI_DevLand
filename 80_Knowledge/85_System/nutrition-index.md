## Notes

1. **Chilean Sea Bass (#18)**: USDA does not list Patagonian toothfish separately. Used "Fish, raw, mixed species, sea bass" as closest proxy. Chilean sea bass is fattier in practice (likely ~120-150 cal, ~8-10g fat per 100g); adjust upward if precision matters.
2. **Microgreens (#42)**: No USDA standard entry exists. Estimated values based on average of common microgreen varieties (broccoli, radish, sunflower sprouts). Actual values vary significantly by type.
3. **Cooking Wine / Shaoxing Wine (#54)**: No USDA entry for Shaoxing wine. Values estimated based on rice wine (sake) nutritional profile with added salt. Sodium varies widely by brand (200-500mg/tbsp).
4. **Chili Oil (#56)**: Values approximate to sesame oil base; capsaicin content is negligible for macros. Some brands add sediment (chili flakes) which adds minimal calories.
5. **Erythritol (#57)**: Listed as 4g carbs per tsp for label accuracy; however, erythritol has ~0.2 kcal/g and is not metabolized as sugar. Net caloric impact is effectively 0.
6. **Shirataki Noodles (#5)**: Values vary by brand. Some brands show 0 calories; USDA reference shows ~9 cal/100g from glucomannan fiber.
7. **Smoked Salmon (#16)**: Very high sodium (2000mg/100g) due to curing process. Monitor portion sizes.
8. **Item 65**: Split into Apple (65a) and Banana (65b) since both were listed.
