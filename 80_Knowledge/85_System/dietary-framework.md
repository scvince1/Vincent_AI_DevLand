# 饮食框架 v1.0

## 基础数据
- 年龄：26（1999-11-03）
- 体重：91kg / 身高：173cm / BMI：30.4
- BMR：1,866 kcal/天（Mifflin-St Jeor）
- TDEE 起步估算：~2,450 kcal/天（轻度活动），需2周体重数据校准

## 核心原则（优先级排序）
1. **蛋白质优先**：145-180g/天（1.6-2.0g/kg）
2. **控制总热量**：TDEE 起步 ~2,450 kcal，减脂期 -300~500 kcal
3. **降低不健康油脂**：减少猪五花/培根频率，优先鱼类、鸡胸、豆腐
4. **降钠**：自己做饭日 <1,500mg，外食日接受 3,000-5,000mg，周均 <3,000mg
5. **保钾**：维持 3,400-4,700 mg/天（不降低）
6. **Na/K 比 <1.0**：核心监控指标

## 每餐追踪五项
热量 / 蛋白质 / 脂肪 / 钠 / 钾

## 实操要点
- 低钠酱油替代普通酱油（879→~550 mg/tbsp）
- 考虑钾盐替代普通盐（75% NaCl + 25% KCl）
- 酱油用量勺量化（1 tsp 减盐酱油 ≈ 180mg Na）
- 每餐蛋白质先行，蔬菜填半盘，碳水灵活
- 一周7天中5天达标即可，不苛求完美

## 饮食偏好与禁忌
- 不吃：蘑菇、木耳及所有菌类/真菌类（恐惧）
- 极少吃：白米、挂面、面包、茄子
- 代糖：赤藓糖醇为主，偶尔木糖醇
- 食材报告默认为 raw 状态
