# Session 2b525bc7-487f-413c-b6fb-76c2920385c2
## Topic
Claude Code datetime hook: 解决 Claude 无法感知当前时间的问题，为公司电脑（受限环境）设计零依赖部署方案，并生成中英文用户手册供 Joyce 及团队使用。

## Facts & Decisions
- Python 3 在 Windows 上被 Microsoft Store 空壳劫持，导致 hook 静默失效
- Vincent 的 Python 安装路径: `C:\Python314\`
- 公司电脑不可下载任何软件或文件，只能手动复制粘贴代码
- 最终选定 JScript (cscript.exe) 方案作为主方案：Windows 原生、零依赖、IT 基本不限制
- 附加 Python 版作为备选
- 文档生成两个版本：中文 + 英文
- 文档卸载章节放在安装前面（先给安全感）
- Step 1 强制"先读代码"（培养代码熟悉度）
- 输出文件: `claude_datetime_hook_zh.md`, `claude_datetime_hook_en.md`
