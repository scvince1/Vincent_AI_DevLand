## Key Statements
- Dump as trust mechanism: removing pre-sorting friction redirects cognitive bandwidth to original thinking and value production
- Polanyi's Paradox manifests concretely in FrontMatter schema design: you cannot fully specify what fields you need before iterating against real material
- Sparse indexing optimizes for discovery; Manifests optimize for lightweight answer retrieval -- different tools for different precision requirements
- Semantic search advantage over RAG for deprecated content contamination: you can locate and remove pollution without rebuilding embeddings
- Pending Terms folder as deliberate contradiction parking: unresolved design contradictions are preserved explicitly, often resolve themselves when new context emerges
- Enterprise three-tier model proposed: personal KB (tacit knowledge) + initiative KB (project-specific) + company RAG (completed initiatives embedded at project end); RAG updates only at initiative start and close
- Token cost as first-class architectural constraint: every design decision maps to token spend; this forces explicitness about what AI actually needs to read
